# NIFTY-50-Investment-Intelligence
AI-powered investment platform built on NIFTY-50 historical data (2000–2021). Features EDA, XGBoost stock prediction with SHAP explainability, Markowitz portfolio optimisation, and risk analytics (Sharpe, VaR, CVaR, Beta, anomaly detection). Built with Streamlit, scikit-learn, and Plotly.
